# fayr_voice_ai

## Environment
- Python > 3.6
- pytorch 1.0
- ubuntu 16.04
- Brain Cloud (Kakaobrain Cluster) V2.XLARGE (2 V100 GPUs, 28 cores cpu, 244 GB memory)

All dependencies and packages can be seen in the Pipfile

Run:

```
$ pipenv --python 3.7
$ pipenv shell
$ pipenv install -r requirements.txt
```
